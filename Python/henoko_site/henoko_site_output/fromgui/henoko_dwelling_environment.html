辺野古の住環境

Dwelling Environment 

辺野古には、居住の場となる賃貸アパートや食料品、雑貨屋のほか、レストラン、カラオケボックスなど、快適に楽しく生活を送る上での施設がそろってきています。 

辺野古公民館

わたしたちのまち辺野古区の中心

賃貸アパート

写真の賃貸アパート以外にもＨＩＧＡ CO‐OP、盛アパート、富士アパート、やぐらビル辺野古などがあります。
アパートの入居等に関するお問い合わせは辺野古公民館では承っておりません。

公園・グランド

海に面し区民の憩いの場となっている辺野古前上原公園や辺野古近隣公園、少年野球にも使われる辺野古グラウンドがあります。

沖縄工業高等専門学校

沖縄工業高等専門学校と学生寮が区内にあります。

小学校・中学校

地元久辺三区の子どもたちが通う、久辺小学校、久辺中学校があります。

商店・レストラン

区民の生活を支える商店やレストランの数々、お近くにお越しの際は是非、お立ち寄りください。
